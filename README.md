## Sobre o projeto

Este é o meu primeiro portfólio pessoal, criado com o objetivo de aprender React na prática. O projeto serviu como campo de estudo para componentes, estado, props e a estrutura de um projeto moderno com TypeScript e Vite.

## Tecnologias

- [React](https://react.dev/) — biblioteca de UI
- [TypeScript](https://www.typescriptlang.org/) — tipagem estática
- [Vite](https://vitejs.dev/) — bundler e dev server
- [Material UI](https://mui.com/) — componentes visuais
- HTML5 e CSS3

## Como rodar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/my-portfolio.git

# Entre na pasta
cd my-portfolio

# Instale as dependências
npm install

# Rode em modo desenvolvimento
npm run dev
```

O projeto estará disponível em `http://localhost:5173`

## Aprendizados

- Criação e composição de componentes React
- Uso de `props` e gerenciamento de `state`
- Tipagem com TypeScript em projetos React
- Utilização da biblioteca Material UI
- Configuração de ambiente com Vite
